PigMC Là Server Nào?

PigMC là một server minecraft mới ở Việt Nam. Sau Bao Nhiêu Năm Di Chuyển Từ PigSMP Thành PigMC, server đã mạnh mẽ cho đến ngày hôm nay! server đã mở thêm nhiều minigames mới như: Bedwars, ArenaPVP, FastBridger, PigArcade... etc. Tham Gia Ngay Hôm Nay Thôi Nào!!!

Máy Chủ Việt Nam

Cách Tham Gia

Các bước để có thể vào được server PigMC.

1

Thêm Máy Chủ

Vào Chế Độ "Multiplayer" Trong Minecraft và thêm server mới
2

Nhập IP Server

Nhập IP: mc.pigsmp.cf Sau đó nhấn "done".
3

Hoàn Tất!

Bây giờ hãy nhấn đúp vào server để khám phá thôi!
